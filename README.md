## Getting Started

### Prerequisites
- Node.js (v16+)
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/seanbud/git-cleanup-PRincess.git
   cd git-cleanup-PRincess
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm run dev
   ```
