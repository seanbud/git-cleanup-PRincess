# Git Cleanup PRincess

Lightweight desktop utility- provides a visual, safe way to audit and prune your changes before they ever hit code review.

- See changes before submitting your PR
- Remove any files you shouldn't have pushed
- Restore any changes you shouldn't have pushed

...so you can keep your kingdom clean and tidy!

### Features
- Safe deletion, files move to the Recycle Bin instead of permanent erasure.
- Network Tree View (similar to git kracken)
- "Prince" mode if you prefer ♂ character.

## Build from Source
If you prefer to build it yourself:
```bash
git clone https://github.com/seanbud/git-cleanup-PRincess.git
cd git-cleanup-PRincess
npm install
npm run dev
```
